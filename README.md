# Spawner
**C# Spawner made for Unity**

Allows you to spawn any GameObject/Prefab with custom rotation and spawn rate (single, burst, infinite).
A bunch of parameters & extras were created but the whole script is still WIP.

## How to use this script :

1. Import this script into your Unity Project.
2. Drag and drop the script in the inspector of the GameObject you want.
3. You can now modiy values in the inspector and test the Spawner.

## How to modify this script :

1. Import this script into your Unity Project.
2. Open the Spawner script in your IDE.
3. Navigate through the code and modify/create/delete anything you'd like to.
